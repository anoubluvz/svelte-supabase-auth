- SvelteJS
- Supabase (Backend)

### Installation:

    git clone https://github.com/syntomy/simpleAuth.git
    cd simpleAuth
    npm i
To run it in dev mode:

    npm run dev
To build it

    npm run build
    
   Build directory is /public/build
